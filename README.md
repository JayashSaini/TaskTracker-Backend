# TaskTracker Backend

This is a backend application for a Task managing app built with TypeScript, Node.js, Express.js, MySQL, and Redis.

## Frontend

Github Repository - [TaskTracker](https://github.com/JayashSaini/TaskTracker-Frontend)

## Tech Stack

- TypeScript
- Node.js
- Express.js
- MySQL
- Redis
